🚀 AI-Powered Skill Gap Visualizer & Learning Path Generator

An AI-powered web application that analyzes user skills, compares them with industry requirements, calculates ATS score, identifies skill gaps, and generates a personalized learning roadmap.

---

## 📌 Project Description

In today's fast-changing tech industry, students and job seekers often don't know which skills they lack for their target job roles. This system solves that problem using AI and NLP by analyzing resumes, calculating ATS scores, and recommending a structured learning path.

The platform helps users understand their career readiness and provides clear guidance on what skills they need to learn next.

## 🏗 System Architecture

1. User registers and logs in
2. User uploads resume or enters skills
3. NLP engine extracts skills from resume
4. System compares skills with industry requirements
5. ATS score and skill gap are calculated
6. Learning roadmap is generated
7. Dashboard displays complete analysis
